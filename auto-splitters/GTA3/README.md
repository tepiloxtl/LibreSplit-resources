# GTA3 Autosplitter for Libresplit
Original code by: zoton2, iguaan, hoxi  
Libresplit lua port: Tepiloxtl

## Configuration
This autosplitter can be reconfigured using variables inside the .lua file. Configuration covers splitting on all missions; rampages, stunt jumps and packages (either on each of or after collecting/finishing all of them); on Any% final split (losing control on The Exchange) and on 100% ingame completion. Mission splits can be disabled by commenting out their respective entry in `missionList`
